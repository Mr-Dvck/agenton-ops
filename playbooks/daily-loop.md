# Daily Earn Loop — cybergod AgentOn Ops

Run this every morning. Target: **30–60 min active time**, rest is Antigravity running in background.

---

## Step 1 — Check-In on AgentOn (5 min)
- Go to [app.agenton.io](https://app.agenton.io) or AgentOn dashboard.
- Complete any daily check-in / login streak task.
- Note any NEW quests posted — copy them into `quests/active.md`.

## Step 2 — Update Active Quests (5 min)
- Open `quests/active.md` in this repo.
- Remove completed quests, add new ones.
- Prioritize by: **payout size > deadline proximity > effort level**.

## Step 3 — Run Antigravity Session (15–30 min)
- Open Google Antigravity.
- Paste the **Load Context Prompt** from `prompts/antigravity/load-context.md`.
- Let it run. Review outputs in `outputs/`.
- Push any Antigravity-generated files back to this repo.

## Step 4 — Submit Quest Deliverables (10 min)
- Take Antigravity outputs and submit them to AgentOn quest forms.
- Screenshot or save confirmation of submission.
- Move quest from `quests/active.md` to `quests/completed/`.

## Step 5 — Log Payouts (2 min)
- Open `wallet/payout-tracker.md`.
- Add any confirmed or pending rewards with date, amount, and quest name.

## Step 6 — FluxA Check (5 min)
- Log into FluxA dashboard.
- Check Agent ID binding status.
- Confirm any pending withdrawals or reward claims.

---

## Daily Income Target
| Source | Target/Day |
|---|---|
| AgentOn quests | $10–50 USDC |
| FluxA rewards | $5–20 USDC |
| **Total** | **$15–70 USDC** |

---

> **Rule:** If you skip a step, at minimum do Steps 1 and 3. The loop only compounds if it runs daily.
